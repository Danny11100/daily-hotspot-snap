# Daily Hotspot Snap

每日全球热点快照 - 基于多维度评分的智能信息筛选工具。

## 核心价值

- **信息降噪**：从每天数百条信息中筛选出10条最有价值的
- **结构化输出**：每条信息包含"来源-解读-影响"三层信息
- **可定制化**：数据源、评分权重、输出格式均可配置

## 适用场景

1. **内容创作者**：获取每日选题灵感，建立个人知识库
2. **产品经理**：跟踪AI、科技领域的产品范式变化
3. **研究人员**：建立信息输入源，节省筛选时间
4. **投资者**：快速掌握影响持仓的全球宏观信号

## 快速开始

1. 安装 skill
```bash
clawdhub install daily-hotspot-snap
```

2. 复制配置
```bash
cp skills/daily-hotspot-snap/config.yaml config.yaml
```

3. 编辑配置（设置你的关注领域、数据源等）

4. 启动自动推送
```bash
openclaw cron enable daily-hotspot-snap
```

## 评分公式

```
总分 = H×0.4 + S×0.3 + R×0.2 + T×0.1

H (热度 40分): Google Trends + 社交媒体讨论
S (情绪信号 30分): 市场波动 + 专家一致性
R (相关度 20分): 与你关注领域的匹配度
T (时效性 10分): 24h内(10分) / 本周(5分)
```

## 工作流程

```
07:30 自动触发
   ↓
抓取数据源（Bloomberg/Reuters/arXiv/HN等）
   ↓
按评分公式计算每条信息得分
   ↓
取前10名高分信息
   ↓
生成热点快照
   ↓
08:10 发送到你的频道
```

## 配置说明

编辑 `config.yaml`：

- **关注领域**：AI、金融科技、Crypto、产品方法论等
- **数据源**：Bloomberg、Reuters、arXiv、Hacker News等
- **评分权重**：热度40%、情绪信号30%、相关度20%、时效性10%
- **输出格式**：字数限制、签名、免责声明
